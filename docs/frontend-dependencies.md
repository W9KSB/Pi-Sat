# Frontend Dependencies

The browser GUI vendors Bootstrap locally so the Raspberry Pi does not need a CDN connection at runtime.

## Bootstrap

- Version: 5.3.3
- Files:
  - `frontend/vendor/bootstrap/bootstrap.min.css`
  - `frontend/vendor/bootstrap/bootstrap.bundle.min.js`
  - `frontend/vendor/bootstrap/LICENSE`
- Source:
  - `https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css`
  - `https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js`
  - `https://github.com/twbs/bootstrap/blob/v5.3.3/LICENSE`
